# Yarn's Sample

https://yarnpkg.com/


# Getting started

```
npm install -g yarn
```

# Check Version

```
yarn --version
```

# Starting a new project

```
yarn init
```

# Adding a dependency

```
yarn add [package]
yarn add --dev [package]
```

# Updating a dependency

```
yarn upgrade [package]
```

# Removing a dependency

```
yarn remove [package]

```

# Installing all the dependencies of project

```
yarn

```
